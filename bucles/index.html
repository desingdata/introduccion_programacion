📚 Introducción a la Programación

Bucles

Los bucles (loops) son estructuras de control que permiten ejecutar un bloque de código de forma repetida mientras se cumpla una condición. Son fundamentales en todo programa.

Empezar a aprender Ir al Playground →

Conceptos Clave

Antes de ver los tipos de bucles, entendamos los elementos que los componen.

🔄

Iteración

Una iteración es cada ejecución del cuerpo del bucle. Si un for-loop corre 5 veces, realiza 5 iteraciones.

❓

Condición

La condición es una expresión booleana que se evalúa antes (o después) de cada iteración. Si es falsa, el bucle termina.

🔢

Variable Contadora

Variable que generalmente se usa para contar iteraciones o recorrer índices. Se inicializa, se verifica y se incrementa en cada ciclo.

♾️

Bucle Infinito

Ocurre cuando la condición del bucle nunca se vuelve falsa. Debe evitarse o manejarse con una instrucción de salida (break).

🛑

break

Instrucción que detiene inmediatamente la ejecución del bucle, sin importar si la condición sigue siendo verdadera.

⏭️

continue

Instrucción que salta la iteración actual y pasa directamente a la siguiente, sin ejecutar el resto del cuerpo del bucle.

Tipos de Bucles

Los tres bucles fundamentales de la programación, con ejemplos comentados en JavaScript y Python.

🔵 for — JavaScript 
// Bucle FOR: ideal cuando conocemos el número
// exacto de iteraciones de antemano

// Estructura: for(inicio; condición; actualización)
for (let i = 0; i < 5; i++) {
  // i++ incrementa el contador en 1 cada vuelta
  console.log(`Vuelta número: ${i}`);
}

// Recorrer un array con for
const frutas = ["manzana", "pera", "uva"];
for (let i = 0; i < frutas.length; i++) {
  console.log(frutas[i]); // accede por índice
}
🔵 for — Python 
# Bucle FOR en Python: itera directamente
# sobre elementos de una secuencia (range, lista)

# range(5) genera: 0, 1, 2, 3, 4
for i in range(5):
    print(f"Vuelta número: {i}")

# Recorrer lista directamente (sin índice)
frutas = ["manzana", "pera", "uva"]
for fruta in frutas:
    print(fruta) # Python simplifica el acceso

# range(inicio, fin, paso): range(0, 10, 2)
# genera pares: 0, 2, 4, 6, 8
🟡 while — JavaScript 
// Bucle WHILE: se ejecuta MIENTRAS la condición
// sea verdadera. Útil cuando no conocemos el
// número exacto de repeticiones

let contador = 0;

while (contador < 5) {
  console.log("Contador: " + contador);
  contador++; // ⚠️ sin esto: bucle infinito!
}

// Ejemplo: pedir entrada hasta que sea válida
let numero = -1;
while (numero < 0) {
  // En práctica real: leer input del usuario
  numero = parseInt(prompt("Ingresa un positivo:"));
}
🟠 do...while — JavaScript 
// DO...WHILE: parecido a while, pero la condición
// se verifica AL FINAL. Garantiza que el bloque
// se ejecute AL MENOS una vez

let intentos = 0;

do {
  console.log("Intento: " + intentos);
  intentos++;
} while (intentos < 3); // verifica después

// Con break y continue:
for (let i = 0; i < 10; i++) {
  if (i === 3) continue; // salta el 3
  if (i === 6) break;   // detiene en 6
  console.log(i); // imprime: 0,1,2,4,5
}
🟢 for...of — JavaScript 
// FOR...OF: itera directamente sobre los VALORES
// de un iterable (array, string, Set, Map).
// Más limpio que el for tradicional con índice

const colores = ["rojo", "verde", "azul"];

for (const color of colores) {
  console.log(color); // rojo, verde, azul
}

// También funciona con strings:
for (const letra of "hola") {
  console.log(letra); // h, o, l, a
}
🔴 for...in — JavaScript 
// FOR...IN: itera sobre las CLAVES (keys) de
// un objeto. Muy útil para recorrer propiedades
// de objetos de JavaScript

const persona = {
  nombre: "Ana",
  edad: 25,
  ciudad: "Bogotá"
};

for (const clave in persona) {
  // clave es: "nombre", "edad", "ciudad"
  console.log(clave + ": " + persona[clave]);
}
// → nombre: Ana
// → edad: 25
// → ciudad: Bogotá

Buenas Prácticas

Consejos esenciales para escribir bucles eficientes y libres de errores.

Evita el Bucle Infinito

Asegúrate siempre de que la variable contadora o la condición cambien con cada iteración para que el bucle termine.

No modifiques el array que recorres

Agregar o eliminar elementos de un array mientras lo recorres puede causar comportamientos inesperados o saltar elementos.

Prefiere for...of sobre for clásico

Cuando solo necesitas los valores (no el índice), for...of hace el código más limpio y legible.

Usa break con precaución

break es útil para salir al encontrar un elemento, pero úsalo con lógica clara para no generar código confuso.

Nombra bien tus contadores

En bucles simples usa i, j. En bucles más complejos usa nombres descriptivos como indice o columna.

Limita los bucles anidados

Los bucles dentro de bucles (anidados) tienen complejidad O(n²). Úsalos solo cuando sea necesario y considera refactorizar.
